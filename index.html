REGRAS E TAGS PARA CRIAÇÃO DE UM DOCUMENTO HTML

  • TODOS AS SINTAXES E TAGS FICAM DENTRO DE "<tag>"
  • TODAS AS SINTAXES E TAGS TERMINAM COM "</tag>"
  • SINTAXES QUE COMEÇAM E TERMINAM NA MESMA TAG SÃO "<tag/>"
  • COMENTÁRIOS SÃO FEITOS COM <!--comentario-->
  • GERALMENTE O CORPO DA PÁGINA É COMPOSTO POR 3 ESTRUTURAS, O HEADER, O MAIN SECTION E O FOOTER (ONDE GERALMENTE TEM INFORMAÇÕES SOBRE O SITE, LINKS DE REDES SOCIAIS ETC.)
  • AS 3 ESTRUTURAS SERÃO ABORDADAS NO 3° ARQUIVO DESSE REPOSITÓRIO

TAGS DE CRIAÇÃO DO DOCUMENTO

  • <!DOCTYPE html> = Indica ao navegador que ele deve ler o documento como um arquivo HTML
  • <html> </html> = é onde vamos armazenar todas as tags e seções da página
  • <head> </head> = onde vamos armazenar informações como a encodificação dos caracteres, titulo da página, descrição da página etc.
  • <meta> = comando usado para definir os metadados da página como a encodificação dos caracteres e descrição da pagina, é usada na seção <head>
  • <title> </title> = onde é armazenado o titulo da página, também usada na seção <head>
  • <body> </body> = é a segunda seção da pagina, é onde vamos armazenar as informações principais da página, e onde as três estruturas (header, main e footer) serão armazenadas

TAGS PARA FORMATAÇÃO DE TEXTO

  • <h1-6> </h1-6> = utilizado para fazer o cabeçalho da página, quanto maior o numero, menor a fonte do cabeçalho
  • <b> </b> = deixa a fonte dentro dele em negrito
  • <i> </i> = deixa a fonte em itálico
  • <p> </p> = faz um paragráfo
  • <hr/> = faz uma linha horizontal para separar o texto
  • <br/> = faz uma quebra de linha
  • <big|small> </big|small> = deixa o texto maior ou menor
  • <sub> </sub> = faz o texto ficar para baixo, como os pequenos numeros usados em elementos quimicos
  • <sup> </sup> = faz o texto ficar para cima, como potencias em um calculo matemático
  • ><pre> </pre> = preserva o jeito que o texto é exibido no código e exibe da mesma forma

TAGS DE CUSTOMIZAÇÃO DE TEXTO (essas tags contém declarações de estilização em CSS, portanto não serão aprofundadas aqui)

todas as sintaxes podem receber um complemento de estilo para mudar suas cores e estilos O complemento é o comando STYLE, um exemplo em que isso seria usado é para mudar a cor do texto, ou de fundo por exemplo:

<h1 style="color: Red"> Teste </h1> ou <body style="background-color: black"> </body>


TAGS PARA ORGANIZAR O CÓDIGO (não são necessárias, porém simplifica o ambiente de trabalho)

  • <header> </header> = mais uma vez utilizado para fazer o cabeçalho da página
  • <nav> </nav> = usado no header como um elemento de navegação, um exemplo é colocar diferentes links usados no site dentro dessa tag

  • <main> </main> = onde o principal conteudo do site costuma estar
  • <article> </article> = usado no main, é feito para fazer um artigo. usado para conter textos, pode conter varios paragrafos, usado para indicar cada artigo de um site

  • <footer> </footer> = o footer é o "final" da página e geralmente contem elementos como informações do website/empresa, ou links paras redes sociais
  • <aside> </aside> = é usado para inserir informações que são exibidas ao usuario, porém não tem ligação direta com o site em questão

TAGS PARA INSERÇÃO DE LINKS, PDFs ou IMAGENS:

  • <a href="link"> </a> = insere um link que leva a outra página, ou site (não abre uma nova aba) pode levar a links de imagens ou pdfs
  • <a href="link" target:"_blank"> </a> = insere um link que leva a outra página, ou site (abre uma nova aba) pode levar a links de imagens ou pdfs
  • <img src="link da imagem ou arquivo na pasta" alt="descrição da imagem"/> = insere uma imagem no site, imagens podem conter links se estiverem entre as tags <a href>, colocar ALT="descrição da imagem" para no caso de a imagem não carregar
  • width/height = definem a largura e altura da imagem, podem ser usados juntos ou apenas 1 deles
  • <video src="link do video ou arquivo na pasta"> </video> = cria uma janela de video, mas só pode ser controlada e reproduzida com a tag "controls". Adicionar "Poster="link ou arquivo"" define uma thumbnail ao video
  • Autoplay e Loop = podem ser adicionados ao video, nomes autoexplicativos
